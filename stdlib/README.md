# Standard Library Layout

The `stdlib/` tree has been created with the initial v0.2 module buckets from the bring-up plan:

- `mem/`
- `io/`
- `strings/`
- `utf8/`
- `fs/`
- `sync/`
- `time/`
- `os/`
- `hal/`
- `testing/`

Implementation work for these modules starts after the core compiler and hosted runtime boundary are stable.
